# Quick Start Guide

Get up and running with KINANA in minutes! This guide provides the essentials to start using the platform right away.

## 5-Minute Setup

### 1. Log In & Set Up Your Profile (1 minute)

- Access your KINANA platform URL
- Log in with your credentials
- Complete your profile (name, preferences)
- Set your timezone and language

### 2. Explore the Four Main Sections (2 minutes)

**Libraries** → Browse and access documents (PDFs, EPUB, Office files)  
**Videos** → Watch educational content and tutorials  
**Podcasts** → Listen to audio content and lectures  
**Brokkly** → Visual coding platform for robotics programming

### 3. Try Brokkly (2 minutes)

1. Click **Brokkly** in the main navigation
2. Select **New Project** or choose a **Starter Activity**
3. Drag code blocks onto the canvas
4. Connect blocks to create a simple program
5. Click **Run** to test (works in simulator mode without hardware)

**Done!** You're now familiar with the basics.

## Your First Project in Brokkly

### Without Hardware (Simulator Mode)

**Quick Activity: LED Blink**

1. Open Brokkly
2. From Activities, select "My First Program" or create a new project
3. Drag these blocks onto the canvas:
   - **Start** block
   - **Turn LED On** block
   - **Wait 1 second** block
   - **Turn LED Off** block
   - **Wait 1 second** block
   - **Loop Forever** block
4. Connect them in sequence
5. Click **Run in Simulator**
6. Watch the virtual LED blink!

**Time: 3-5 minutes**

### With Hardware (Connected Kit)

**Prerequisites:**

- One of our robotics kits (Maker, RC Car, Discovery V3, or Dualsense)
- USB cable or Bluetooth connection
- Kit powered on

**Quick Activity: Hello Hardware**

1. Connect your kit via USB or Bluetooth
2. In Brokkly, click **Connect Hardware**
3. Select your kit type
4. Wait for connection confirmation (green indicator)
5. Load any "Getting Started" activity for your kit
6. Click **Upload to Device**
7. Watch your kit respond!

**Time: 5-10 minutes**

## Essential Features at a Glance

### Libraries

**Upload a Document**

1. Go to Libraries
2. Click **Upload** or drag-and-drop
3. Add title and description
4. Save

**Find Content**

- Use the search bar (top right)
- Apply filters (format, date, category)
- Browse collections and folders

### Videos

**Watch a Video**

- Click Videos → Browse or search
- Click any thumbnail to play
- Use playback controls (speed, quality, subtitles)

**Upload Your Own** (if you have permissions)

- Click Upload → Choose file or paste URL
- Add metadata (title, description, tags)
- Publish

### Podcasts

**Stream Audio**

- Navigate to Podcasts
- Select an episode or series
- Click play
- Adjust playback speed if desired

**Download for Offline**

- Click the download icon next to any episode
- Access from your downloads folder

### Brokkly Visual Coding

**Code Block Categories**

- **Control**: Start, Stop, Loop, If/Else
- **Motors**: Move forward/backward, turn, speed control
- **Sensors**: Read distance, light, touch, color
- **Logic**: Compare values, math operations, variables
- **Actions**: LED, sounds, displays, servos

**Workspace Tips**

- **Drag & Drop**: Pull blocks from the left panel to canvas
- **Connect**: Snap blocks together (they'll highlight when close)
- **Delete**: Drag unwanted blocks to trash or press Delete
- **Zoom**: Use mouse wheel or zoom controls
- **Pan**: Click and drag on empty canvas space

## Quick Tips

**Navigation**  
Main menu is always accessible - click the hamburger icon or logo to return home.

**Search Everything**  
The global search (top right) searches across all content types.

**Save Often**  
In Brokkly, your project auto-saves, but use **Save Project** before closing.

**Use Activities**  
Don't start from scratch - Activities provide ready-to-use templates and lessons.

**Test in Simulator**  
No hardware? Use the simulator to test all your Brokkly code.

**Hardware Connection**  
Green icon = connected, Yellow = connecting, Red = disconnected

**Keyboard Shortcuts** (Brokkly)

- `Ctrl/Cmd + S`: Save project
- `Ctrl/Cmd + Z`: Undo
- `Ctrl/Cmd + Y`: Redo
- `Delete`: Remove selected block
- `Ctrl/Cmd + C/V`: Copy/Paste blocks

## Troubleshooting Quick Fixes

**Can't log in?**  
→ Check credentials, reset password if needed, contact admin

**Hardware won't connect?**  
→ Check power, USB connection, try different USB port, restart kit

**Code won't upload?**  
→ Verify hardware connection (green icon), check for block errors (red highlights)

**Video won't play?**  
→ Check internet connection, try different browser, clear cache

**Can't find content?**  
→ Use search, check filters aren't too restrictive, verify access permissions
